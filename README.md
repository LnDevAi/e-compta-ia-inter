# E-Compta IA Inter

Plateforme de comptabilité intelligente inter-entreprises propulsée par l'IA — automatisation des écritures, rapprochement bancaire et reporting financier.

## Branches

| Branche | Rôle |
|---------|------|
| `main` | Production stable |
| `dev` | Intégration continue |
| `feature/frontend-neya` | Développement UI/UX |
| `feature/security-zombre` | Sécurité & authentification |

## Stack

- Frontend : Vue.js / Nuxt
- Backend : Python / FastAPI
- IA : OpenAI API / LangChain
- Base de données : PostgreSQL + Redis

## Démarrage rapide

```bash
git clone https://github.com/LnDevAi/e-compta-ia-inter.git
cd e-compta-ia-inter
cp .env.example .env
pip install -r requirements.txt
uvicorn main:app --reload
```
